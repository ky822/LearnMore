### 符号

Ruby解释器会维护一个符号表（Symbol Table），而Ruby程序中使用符号的意义在于，通过将开销较大的字符串操作转化成了一个开销较小的整数操作。

* 定义？
* 调用

        name = :size

* String 和 Symbol 相互转化
    * String 到 Symbol：`intern` 或 `to_sym`
    * Symbol 到 String : `id2name` 或 `to_s`
