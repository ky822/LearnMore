### 定义简单的方法

全局方法，不用在类的实例基础上就可以调用的方法，和实例方法相对。举例如下：

    def factorial(n)
        if n < 1
            raise "argument must be > 0"
        elsif n == 1
            1
        else
            n * factorial(n - 1)
        end
    end
