### 1. 控制语句
##### 1.1 if-else语句

    if (r == 4) {
        x <- 1
    } else {
        x <- 3
        y <- 4
    }
    
##### 1.2 循环结构

    while (condition) {
        statements
    } 
    ---
    repeat {
        statements
    }
    ---
    break        # 跳出当前循环
    next         # 跳出本次循环，继续下次循环
